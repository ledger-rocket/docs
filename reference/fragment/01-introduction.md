# Fragment Ledger API - Introduction

## What is Fragment?

Fragment is a comprehensive toolkit designed for building financial products that move and track money. The platform provides both an API and Dashboard, enabling developers to design, implement, and operate ledger systems.

## Core Features

The documentation covers several key functional areas:

### Ledger Design and Management

- Setting up ledger accounts
- Posting ledger entries
- Grouping entries for organization
- Reconciling payments through links

### Implementation and Deployment

- Dashboard-based configuration
- API integration
- CLI tooling
- SDK support for TypeScript, Python, Go, and Ruby

### Financial Operations

- Reading account balances with multiple consistency modes
- Querying ledger data and generating reports
- Handling multi-currency transactions
- Managing data migrations and schema updates
- Exporting data to external systems like S3 and Retool

### Advanced Capabilities

- Entry reversals and reposting
- Payment synchronization with external providers (Increase, Stripe, Unit)
- Custom payment link integration
- Metadata storage through entry tags
- Consistent group management

## Documentation Structure

The resource is organized into logical sections covering design, deployment, data operations, and system configuration, with accompanying API reference materials and a changelog tracking feature releases.
