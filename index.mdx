## Welcome to LedgerRocket

LedgerRocket is a real-time core ledger and finance platform for financial services via API. It centralizes transaction events into a single, strongly consistent, double-entry ledger with an immutable audit trail and matching accounting treatments.

## Key Capabilities

- **Efficiency** — single source of truth for all transaction data and business events
- **Auditable** — fully traceable audit trail back to every business event, in real-time
- **Compliance** — accounting rules built directly into the processing flow
- **Speed** — real-time balances, zero batch windows, ultra-fast processing
- **Composable** — components available as a whole integrated platform or individually
- **Source-available** — software available as a source-available licence

## Architecture

LedgerRocket follows a layered architecture from upstream systems through to downstream consumers:

| Service | Role |
|---------|------|
| **Reference Data Service** | Sites, entities, ledgers, accounts, account codes |
| **Accounting Rule Engine** | Template lifecycle — authoring, validation, maker-checker approval |
| **Event Service** | Atomic event processing — validates events, evaluates templates, creates transfers |
| **Balance Cache** | Balance queries — real-time, historical, chain analysis |
